Section 2 notes

  • Pages should have only one main heading element (h1), but may have multiple lower-level headings.
  • Use <strong> and >em< instead of <b> and <i> for semantic meaning.
  • Always include the attribute "alt" when using images.
  • Complete, self-contained elements can be represented using the <article> tag. Examples: itens of an e-commerce, blog posts, etc.
